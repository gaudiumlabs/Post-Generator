If you want a more in-depth tutorial , check out:
http://www.vogella.com/tutorials/MySQLJava/article.html


1. Need community edition free version, which can be obtained from the oracle mysql website
Gaudium has a user account on oracle. Email is username and password is known.
	
	https://dev.mysql.com/downloads/mysql/

2. Go through the installation process. Install as a “developer” You don’t need visual studio or python 3.4, unless you are developing with those.
A new service will be created for MYSQL, the service is MYSQL56
MySQL workbench. This is a great way to see the tables , schema and data.

3)Java JDBC connection/interface:
	i) Setup:  Download the connector from 
		http://dev.mysql.com/downloads/connector/j
	ii) In the installation directory (where mysql is installed), you will find
		the JAR file needed. Put this in the "lib" folder under the subfolder "SQLrelated"

AND...... you're done. Tadaaa!
